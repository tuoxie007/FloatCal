# FloatCal

一个轻量的 macOS 悬浮计算器。

## 功能

- 全局快捷键呼出/隐藏，默认 `Command + 1`
- 支持自定义快捷键
- 菜单栏常驻入口
- 支持键盘直接输入
- 计算器窗口默认居中显示

## 使用

1. 打开 `FloatCal.xcodeproj`
2. 运行 `FloatCal`
3. 按 `Command + 1` 呼出计算器
4. 点击左上角齿轮可修改快捷键

## 键盘支持

- 数字：`0-9`
- 运算：`+` `-` `*` `x` `/` `%`
- 括号：`(` `)`
- 小数点：`.`
- 计算：`Enter`
- 删除：`Delete`
- 清空：`Esc` / `c`
